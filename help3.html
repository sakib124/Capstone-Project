Introduction

Introduction

This help page is a guide to help use the Manage Patient Page. There are a number of different functionalites used on this page, such as Copy, Move, Delete, and Play. All of these functioanlities will be discussed in depth

Copy

The Copy button allows users to copy media files into different albums. This will save the media files both in target album and original album

  1. Users selects media files by the checkboxes to the left of each media file
  2. User selects album to copy to from drop-down menu entitled "Album to move/copy to"
  3. Users presses Copy button

Move

The Move button allows users to moves media files into different albums. This will not save the media files in the originial album, but instead move selected media files into designated new album

  1. Users selects media files by the checkboxes to the left of each media file
  2. User selects album to move to from drop-down menu entitled "Album to move/copy to"
  3. Users presses Move button

Delete

The Delete button will delete media files from the data base

  1. Users selects media files by the checkboxes to the left of each media file
  2. Users presses Delete button

Play

The Play button will play media files. Users will be redirected to the Slide Show Page

  1. Users selects media files by the checkboxes to the left of each media file
  2. Users presses Play button
